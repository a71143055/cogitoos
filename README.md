# CogitoOS (Python Prototype)

"나는 생각한다, 고로 존재한다"를 운영체제 개념으로 모사한 파이썬 서비스.
- Intent(의도) 우선
- Self-Model(자기모델) 기반 정책
- Memory & Trace(이유 트레일)
- Privacy Governor(경계)
- Experience Shell(목표 중심 UI, 콘솔)

## Requirements
- Python 3.11 (>=3.10)
- No external packages

## Run
```bash
python -m cogitoos.main
python -m pytest
